# Socket OOB Control Message Utils for Go

> Experimental

## Planned features
- append, read, modify, remove
  - generic socket OOB control messages
  - ECN flags
  - GSO size
  - IPv6 destination options

## Supported Systems
Currently only Linux is supported.
